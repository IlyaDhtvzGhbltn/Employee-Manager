Необходимо реализовать простое приложение на WPF. Суть приложения - управление списком сотрудников (ФИО). 
В приложении должен отображаться :
1)Cам список, 
)Kнопки удаления и добавления. 
При первом запуске приложения,  список должен заполняться начальными данными.
Можно взять любой список, желательно более 100 записей

Реализовать простую  форму авторизации,  предварительно создав 2х пользователей admin и user. !!!
Предусмотреть возможность расширения количества полей, отображаемых в списке.
Например позже может добавиться поле "номер телефона".

Если пользователь является администратором, разрешить редактировать список сотрудников с сохранением изменений в базу данных (изменять,  добавлять и  удалять записи).
Если пользователь не администратор, скрыть элементы управления для редактирования списка.
При добавлении нового сотрудника приложение должно проверять запись на уникальность, в случае нахождения сотрудника с таким же ФИО - выдавать ошибку.
Также необходимо реализовать строку поиска по фамилии, которая отфильтровывает основной список.
Дизайн окна - на свое усмотрение. Ключевые моменты:
 1 - Использование паттерна  MVVM и async\await.
 2 – Использование любого IoC для разделения абстрактной части от реализации.  
Дополнительно предусмотреть возможность смены цветовой схемы окна.

Хранение данных – локальная база MsSql. Использование ORM - на свое усмотрение.
Клиент может сам подключатся к БД.
